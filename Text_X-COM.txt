Курсовая работа выполнена с целью решить практическое задание по дисциплине «Информационные технологии» на языке C# с использованием Entity Framework Core в среде программирования Microsoft Visual Studio. Приложение создано для продажи товаров магазина электроники, а также для поиска информации о них.
Задание направлено на реализацию приложения с удобным графическим интерфейсом, которое предназначено как для работников магазина, так и для покупателей. Хранятся данные о товарах, покупателях, персонале и акциях.
Приложение позволяет:

Просмотр и ознакомление с каталогом товаров
Выбор и покупка товаров
Создание акций, добавление и редактирование информации о товарах.
Ключ для входа в админ-меню 12345, там управление товарами осуществляется и просмотр информации о клиентах, покупках и отзывах
